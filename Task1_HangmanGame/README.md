## Hangman Game

This project is a **console-based Hangman game** developed using **Python** as part of the **CodeAlpha Python Programming Internship**.

### Features:
- Random selection of words for gameplay
- User guesses letters one at a time
- Limited chances for incorrect guesses
- Displays win or game-over messages

### Tools Used:
- Python
- Visual Studio Code

### Project Files:
- `hangman.py` – Python source code
- `output.png` – Screenshot of program execution
- `README.md` – Project documentation

### How to Run:
1. Open the project folder in VS Code
2. Run the file using `python hangman.py`
3. Follow the on-screen instructions to play the game

### Learning Outcome:
- Understanding loops and conditional statements
- Working with lists and user input
- Implementing basic game logic in Python
